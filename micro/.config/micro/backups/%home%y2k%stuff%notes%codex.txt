control reference:

hyprland / desktop controls:

 when navigating Y2KOS, the main modifier (the equivalent to the windows or fn key)
 is Alt or Caps Lock (multiple options for convenience depending on the keybind or your hands)
 some technical details get a bit finnicky, but if you'll humor me i can explain.

 when setting the main modifier key in hyprland, you are only allowed to set it to one key.
 This would be fine, but only certain functions support multiple inputs *natively* in hyprland, so i have a workaround.

 first, was to set the main modifier key to be the windows key in hyprland.conf (but we wont actually be using it as intended)

 $mainMod = SUPER # set wink as main modifier (which has been swapped to alt and caps)
 
 next, in the input variables of hyprland.conf, i set the following options:

 input {
     kb_options = caps:super, altwin:swap_alt_win
 }

 there are other options between, but whats shown is all that's needed. 
 these options set the caps lock key to the windows key, and *swaps* the alt and windows key.

 as a finished result, the windows key on the keyboard functions as the usual alt, 
 while the caps lock and alt keys now interchangably work as the main modifier, depending on your preference.

 now with that out of the way, heres the real binds list:
 
 Remember, Mod just means Caps Lock or Alt :)
 Also, ALL binds are configurable in /$HOME/.config/hypr/hyprland.conf!     **** VERIFY THIS FUTURE MORGAN!!!! ****
